# Level-statistics-of-the-disordered-Haldane-Shastry-model-with-1-r-alpha-interaction
This repository contains the source codes for Exact Diagonalization (ED) of the generalized generalized Haldane-Shastry model with $1/r^{\alpha}$ interaction, the datasets for the mean gap ratio, and the Jupyter notebooks used to reproduce the plots presented in our paper
